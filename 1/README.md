# دعوة فرح #1 — أحمد و هايدي

> اللينك المرجعي: <https://ahmed-haidy-18-5-2026.vercel.app/>

## 💍 الدعوة بتتكلم عن إيه؟

دعوة زفاف **أحمد و هايدي** يوم **١٨ / ٥ / ٢٠٢٦** الساعة **٧:٠٠ مساءً** في
**فندق صن رايز ازور سابقاً — سيدي جابر، الإسكندرية** (Sunrise Alex Avenue Hotel).

نص الدعوة على الموقع: *"بداية فصلٍ جديد من رحلتنا — يسعدنا حضوركم."*

## 🎨 إيه اللي عملته في النسخة دي

اتعمل scrape للموقع الأصلي بـ Playwright (لأنه React SPA)، واتجابت كل الأصول
(الفيديو، الموسيقى، الصور) محلياً، واتبني نسخة Static **بنفس الشكل والترتيب
والتجربة** بدون أي اعتماد على بناء/سيرفر.

### الأقسام بالترتيب
1. **Intro Overlay** — فيديو خلفية + كارت "ابدأ الدعوة ✦".
2. **Curtain Reveal** — انتقال بستارة كحلي مع علامة ✦ ذهبية.
3. **Hero** — صورة العروسين داخل إطار ذهبي + جملة "بداية فصلٍ جديد من رحلتنا".
4. **Names + Countdown** — كارت رخامي فيه الأسماء، نص الدعوة، عدّاد تنازلي
   لحظي (ثانية : دقيقة : ساعة : يوم) + التاريخ والساعة والمكان.
5. **Story** — ٣ كروت: اللقاء الأول، الخطوبة، يوم الزفاف.
6. **Family** — رسالة شكر للعائلة.
7. **Location** — خريطة Google Maps مدمجة + زر Location يفتح اللوكيشن.

### المزايا
- 🎬 فيديو افتتاحي يشتغل Auto-play (muted).
- 🎵 زرار موسيقى خلفية أعلى-يسار، شغّال/إيقاف.
- ⏳ عدّاد تنازلي مظبوط على **2026-05-18 19:00 (+03:00)**.
- 🗺️ خريطة مدمجة + زرار يفتح اللوكيشن الأصلي على Google Maps.
- 📱 Responsive كامل (موبايل / تابلت / ديسكتوب) ودعم RTL.
- 🎨 نفس الـ Color palette الأصلي: ذهبي (`hsl(43 65% 52%)`) + كحلي
  (`hsl(217 69% 14%)`) + عاجي (`hsl(39 33% 96%)`).

### التكنولوجيا
- **HTML5 + CSS3 + Vanilla JavaScript** — zero dependencies.
- **خطوط:** Amiri (للعناوين) + Tajawal (للنصوص) من Google Fonts.
- **التصميم:** أعيد بناؤه على شكل CSS عادي (مش Tailwind) عشان يشتغل من غير build.

## ▶️ طريقة التشغيل

افتح `1/index.html` بدبل كليك، أو شغّل سيرفر محلي من جذر الفولدر:

```powershell
# من داخل فولدر 1
python -m http.server 8080
# ثم افتح http://localhost:8080
```

> ملاحظة: تشغيل الموسيقى تلقائياً ممنوع من المتصفحات قبل أي تفاعل، لكن لما
> تضغط "ابدأ الدعوة ✦" الموسيقى بتشتغل مع الانتقال.
